2. 你所使用含有 webpack 的项目中，webpack 是通过 webpack-cli 打包的吗？

   是，nextjs 和 umijs 2.直播后明白并不是，nextjs 和 umijsj 将 webpack 包直接放入 compiled 文件夹 ，防止依赖更新带来的问题，所以是 api 打包的
